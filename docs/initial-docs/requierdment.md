🎯 プロジェクト目標
配当金投資家が複数の証券口座を一元管理し、投資回収率と予想配当を可視化できるWebアプリケーションを構築する。

👥 想定ユーザー
配当金投資を行っている個人投資家
複数の証券口座を保有している
日本株・米国株・ADRに投資している
投資回収率を定量的に把握したい

🧮 計算ロジック
投資回収率
投資回収率 = (累積配当額 ÷ 投資額合計) × 100
税金計算
日本株（特定口座）:

税引後 = 税引前 × (1 - 0.20315)
米国株（特定口座）:

外国源泉税 = 税引前 × 0.10
日本所得税 = (税引前 - 外国源泉税) × 0.20315
税引後 = 税引前 - 外国源泉税 - 日本所得税
NISA:

日本株: 税引後 = 税引前（非課税）
米国株: 税引後 = 税引前 × (1 - 0.10)（外国源泉税のみ）
為替換算
円換算額 = ドル金額 × USD/JPY為替レート
大体入金額
大体入金額 = 投資額合計 - 累積配当額合計

ExchangeRate-API（自動取得）
取得データ:

USD/JPY為替レート（1日1回）
手動入力（必須）
入力データ:

保有銘柄情報（銘柄、株数、平均取得単価）
配当受取実績（受取日、税引後金額）
理由:

証券会社ごとに税金計算が微妙に異なる
為替レート（確定時点）が各社で異なる
完全自動化は困難
